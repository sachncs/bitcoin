# Support

## Where to get help

- **Documentation** — See the [README](README.md) and [`docs/`](docs/)
  directory for API reference, architecture overview, and usage examples.
- **Bug reports / feature requests** — Open a [GitHub
  Issue](https://github.com/sachn-cs/bitcoin/issues) with a clear description
  and reproduction steps.
- **Security vulnerabilities** — See [SECURITY.md](SECURITY.md) for private
  disclosure instructions.
- **Discussions / questions** — Use [GitHub
  Discussions](https://github.com/sachn-cs/bitcoin/discussions) for general
  questions, ideas, and community conversation.

## Before opening an issue

1. Search [existing
   issues](https://github.com/sachn-cs/bitcoin/issues?q=is%3Aissue) to see if
   your question has already been asked.
2. Check the [CHANGELOG](CHANGELOG.md) to see if your concern has already been
   addressed in a newer version.
3. Make sure you are using the latest release: `pip install --upgrade bitcoin`.
